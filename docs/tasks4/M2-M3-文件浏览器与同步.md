## M3: 后端文件同步协议

### 后端 — FileSync WebSocket Handler

`backend/app/tools/editor_sync.py`：
- `list_files(path)` → 目录树 JSON
- `read_file(path)` → 文件内容 + 元数据
- `write_file(path, content)` → 保存（原子写入）
- `create_file(path, type)` → 新建
- `delete_file(path)` → 删除
- `rename_file(old, new)` → 重命名
- `search_files(query)` → 语义搜索

### WebSocket 协议（与编辑器同步）

```json
// 编辑器 → 后端
{"type": "fs_list", "path": "/src"}
{"type": "fs_read", "path": "/src/main.py"}
{"type": "fs_write", "path": "/src/main.py", "content": "..."}

// 后端 → 编辑器
{"type": "fs_list_result", "files": [...]}
{"type": "fs_content", "path": "/src/main.py", "content": "...", "language": "python"}
{"type": "fs_saved", "path": "/src/main.py"}
```

### 安全

- 路径沙箱：限制在项目根目录
- 路径遍历防护
- 大文件分片读取（>1MB）
