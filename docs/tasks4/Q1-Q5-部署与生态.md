# Phase Q: 部署与生态

> **总预估**：4.5 周 | **前置**：无

## Q1: Docker Compose 一键部署（1 周）

`docker-compose.yml`：

```yaml
services:
  minicc-backend:
    build: ./backend
    ports: ["8000:8000"]
    volumes: ["./workspace:/workspace"]
    env_file: .env

  minicc-frontend:
    build: ./frontend
    ports: ["3000:3000"]
    depends_on: [minicc-backend]

  chrominium:  # for Playwright RPA
    image: browserless/chromium
    ports: ["3001:3000"]

  # 可选
  redis:
    image: redis:7
    ports: ["6379:6379"]

  searxng:  # 可选：自托管搜索
    image: searxng/searxng
    ports: ["8888:8080"]
```

## Q3: 插件市场（1 周）

`backend/app/marketplace/`：

```python
class PluginPackage(BaseModel):
    id: str
    name: str
    version: str
    type: Literal["agent", "rule", "tool", "workflow", "skill"]
    description: str
    manifest: dict  # 插件描述文件
    download_url: str
```

**功能**：
- 从 GitHub 仓库同步插件列表
- 插件搜索和安装
- 版本管理和更新
- 依赖解析
- 沙箱执行（插件权限隔离）

## Q5: CI/CD 集成（0.5 周）

GitHub Actions 模板：

```yaml
name: MiniCC CI
on: [pull_request]

jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Code Review
        run: |
          curl -X POST https://minicc.example.com/api/cicd/review \
            -H "Authorization: Bearer ${{ secrets.MINICC_TOKEN }}" \
            -d '{"pr_id": ${{ github.event.pull_request.number }}}'
```
