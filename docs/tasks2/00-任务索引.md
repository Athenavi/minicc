```
                   ├── C2 Word
                   └── C3 Email
                           │

                    ├── B2 OCR

                    ├── D2 Executor ── 依赖 D1

                    └── D6 UI Editor ── 依赖 D1

                    └── E2 SelfHeal ── 依赖 D2
                           │

                    └── F2 API
```

> **推荐开发顺序**：A1 → A2/A3 → C1 → B1 → D1/D2 → D6 → E1/E2 → F1/F2
