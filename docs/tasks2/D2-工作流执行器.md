## 2. 详细子任务

### 2.1 WorkflowExecutor

- [ ] `automator/workflow.py`

```python
class WorkflowExecutor:
    """
    工作流执行器。解析 DSL 并按步骤执行。
    - 顺序执行步骤
    - 条件分支 (if/else)
    - 循环 (loop over)
    - 变量替换
    - 错误处理与重试
    - 超时控制
    - 状态持久化
    """
    
    async def execute(
        self,
        workflow: WorkflowDefinition,
        context: WorkflowContext,
    ) -> WorkflowResult:
        ...
```

### 2.2 执行流程

```
1. 解析 DSL → WorkflowDefinition
2. 初始化 WorkflowContext（变量、env、内置对象）
3. 按顺序执行 steps:
   a. 变量替换 params
   b. 检查 if 条件
   c. 调用对应 Tool
   d. 更新上下文（步骤结果存入 .steps.{id}.result）
   e. 处理循环
   f. 错误处理（重试/跳过/通知）
4. 返回 WorkflowResult（状态、各步骤结果、错误信息）
```

### 2.3 SSE / Broadcaster 集成

- 每个步骤开始/完成时发送事件
- 实时进度推送
- 步骤耗时报告

### 2.4 状态持久化

- 执行中的工作流状态保存到 SQLite
- 支持中断后恢复
- 执行历史可查询
