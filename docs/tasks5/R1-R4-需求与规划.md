# R1-R4: AI 产品经理 — 需求与规划

> **总预估**：3.5 周 | **产出**：PRD → 技术方案 → 任务拆解 → 确认

## R1: AI PRD 生成器（1 周）

`backend/app/pm/prd_generator.py`：

```python
class PRDGenerator:
    """从自然语言生成结构化 PRD。"""

    async def generate(self, description: str, context: dict = None) -> PRD:
        """生成包含以下部分的需求文档：
        - 产品概述（背景、目标用户、核心价值）
        - 功能列表（优先级 MoSCoW）
        - 用户故事（As a... I want... So that...）
        - 验收标准（Given... When... Then...）
        - 边界条件
        - 非功能需求（性能、安全、可用性）
        """
```

### 输出格式

```markdown
# PRD: [项目名称]

## 产品概述
- 背景：
- 目标用户：
- 核心价值：

## 功能列表
| 功能 | 优先级 | 描述 | 验收标准 |
|:-----|:-------|:-----|:---------|
| ...  | P0     | ...  | ...      |

## 用户故事
- As a [角色], I want [功能], So that [价值]

## 非功能需求
- 性能：
- 安全：
- 可用性：
```
