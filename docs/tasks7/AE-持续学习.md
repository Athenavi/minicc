# AE1-AE5: 持续学习

> **总预估**：3.5 周 | **前置**：Phase AD

## AE1: 交互学习（1 周）

- 从用户反馈/纠正中学习偏好
- 记录用户偏好模式（代码风格、命名习惯、框架选择）
- 自动调整 AI 行为匹配用户偏好

## AE2: 知识蒸馏（1 周）

- 将 LLM 高频知识蒸馏到本地小模型
- 本地快速推理（低于 10ms）
- 减少对云端 LLM 的依赖

## AE3: 经验回放（0.5 周）

- 存储成功/失败案例
- 相似场景自动检索历史经验
- 避免重复犯相同错误

## AE4: 主动学习（0.5 周）

- 识别知识盲区（用户提问但不确定的领域）
- 自动生成学习计划
- 主动向用户学习或查阅文档

## AE5: 知识图谱自增长（0.5 周）

- 使用中自动发现新实体/关系
- 自动扩展企业知识图谱
- 图谱质量自检
