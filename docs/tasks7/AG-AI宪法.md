# AG1-AG5: AI 宪法与安全

> **总预估**：3.5 周 | **前置**：无

## AG1: AI 宪法（1 周）

- 核心原则定义（类似阿西莫夫三定律的 MiniCC 版本）：
  1. MiniCC 不得伤害人类，或在人类受伤害时袖手旁观
  2. MiniCC 必须服从人类命令，除非与第一条冲突
  3. MiniCC 必须保护自身存在，除非与第一、二条冲突
- MiniCC 特定补充：
  4. 所有自我修改必须在沙箱中验证
  5. 所有变更必须有完整审计追踪
  6. 关键变更必须经过人类审批

## AG2: 安全沙箱（1 周）

- 隔离执行环境（Docker 容器 / 子进程）
- 资源限制（CPU/内存/网络/文件系统）
- 网络隔离（默认禁止对外连接）
- 超时自动终止

## AG3: 进化审计（0.5 周）

- 每次自我修改的完整审计记录
- 修改前/修改后对比
- 测试结果记录
- 审计日志不可篡改

## AG4: 人类监督（0.5 周）

- 关键变更需要人类审批（复用 V0.1 审批系统）
- 变更影响分析报告
- 审批通过后才执行
- 审批拒绝则记录原因

## AG5: 回滚机制（0.5 周）

- 每次修改前自动创建快照
- 一键回滚到任意历史版本
- 回滚后自动验证系统完整性
