# AF1-AF5: 物理世界融合

> **总预估**：5 周 | **前置**：无

## AF1: 机器人控制（1.5 周）

- ROS2 集成（话题/服务/动作通信）
- 机械臂控制接口（MoveIt2）
- AGV/无人机导航接口
- 安全协议：速度/力矩限制 + 急停

## AF2: IoT 网关（1 周）

- MQTT Broker 集成
- 设备注册/发现/管理
- 传感器数据采集与处理
- 设备控制指令下发

## AF3: 边缘计算（1 周）

- MiniCC Edge（Raspberry Pi / Jetson 运行）
- 本地推理（复用 AE2 蒸馏模型）
- 断网离线运行
- 云端同步（可选）

## AF4: 数字孪生（1 周）

- 物理设备的实时 3D 数字镜像
- 传感器数据 → 数字模型同步
- 模拟仿真（在数字孪生中测试控制策略）

## AF5: 自动化实验室（0.5 周）

- AI 自主设计并执行物理实验
- 实验数据自动采集分析
- 实验结果 → 知识图谱
