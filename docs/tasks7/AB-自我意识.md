# AB1-AB5: 自我意识 — 元认知系统

> **总预估**：4.5 周 | **前置**：无

## AB1: 自我监控器（1 周）

`backend/app/self/self_monitor.py`：
- 实时追踪：CPU/内存/响应时间/QPS/错误率
- Prometheus 指标暴露
- 异常检测（基线与实时对比）
- 历史趋势存储

## AB2: 架构分析器（1 周）

- 代码结构分析：模块依赖图、耦合度、内聚性
- 技术债务估算：TODO/FIXME/HACK 密度
- 循环依赖检测
- 输出：架构健康报告

## AB3: 性能剖析（1 周）

- 热点路径追踪（请求链路分析）
- 数据库慢查询识别
- 内存泄漏检测
- 输出：性能优化建议列表

## AB4: 进化规划器（1 周）

- 综合 AB1-AB3 数据 → 生成改进优先级
- 自动生成改进计划（做什么、怎么做、预期效果）
- 计划版本管理

## AB5: 健康仪表盘（0.5 周）

- 前端：自身状态可视化（类似人类体检报告）
- 分数：架构健康分 / 性能分 / 安全分
- 趋势图
