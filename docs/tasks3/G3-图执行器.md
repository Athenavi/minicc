```python
class GraphExecutor:

    - invoke(graph, initial_state) → final_state

        """同步执行整个图。"""
        ...
    
    async def astream(self, graph: StateGraph, state: dict):
        """流式执行，每个节点完成时 yield 事件。"""
        ...
```

### 2.2 执行流程

```
1. 拓扑排序 nodes
2. 初始化状态 (state = initial_state | schema_defaults)
3. 执行 entry_point 节点:
   a. 根据 node_type 路由到对应执行器
   b. 更新状态
   c. 检查出边条件
   d. 激活下一批节点
4. 并行分支同时执行 (asyncio.gather)
5. 重复直到无更多可执行节点
6. 返回最终状态
```

## 3. 验收标准

| # | 检查项 | 验证方式 |
|:-|:-------|:---------|
| 1 | 线性图顺序执行 | 3 个节点按顺序执行 |
| 2 | 条件分支正确执行 | 条件满足走 A，否则走 B |
| 3 | 并行节点同时执行 | 总执行时间 ≈ 最慢节点时间 |
| 4 | 节点失败处理 | 失败节点标记 error，其他继续 |
| 5 | 流式输出 | 每个节点完成时触发 event |
