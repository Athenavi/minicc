# H1：文档加载器

> **所属阶段**：Phase H - RAG 管道
> **预估工时**：3 天

---

## 1. 任务目标

实现多源文档加载器，支持 PDF/HTML/Office/Web 页面/代码库的文本提取。对标 Dify 的文档加载能力。

## 2. 详细子任务

```python
# backend/app/rag/loader.py

class Document(BaseModel):
    content: str
    metadata: dict[str, Any] = {}
    source: str = ""

class DocumentLoader(ABC):
    @abstractmethod
    async def load(self, source: str) -> list[Document]: ...
```

### 2.1 加载器实现

- [ ] `PDFLoader` — PyMuPDF 提取文本 + 元数据
- [ ] `HTMLLoader` — BeautifulSoup 提取正文
- [ ] `OfficeLoader` — python-docx / openpyxl 提取
- [ ] `WebPageLoader` — Playwright 渲染后提取
- [ ] `CodeLoader` — 递归加载代码目录（跳过 .git/node_modules）
- [ ] `TextLoader` — 纯文本/代码文件

### 2.2 多源聚合
