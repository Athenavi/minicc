## рҹҡҖ еҝ«йҖҹејҖе§Ӣ

### еүҚжҸҗ
- Python 3.14+
- Node.js 22+
- (еҸҜйҖү) Redis 7+

### е®үиЈ…

```bash
# е…ӢйҡҶ
git clone https://github.com/Athenavi/minicc.git
cd minicc

# еҗҺз«Ҝ
cd backend
pip install -e ".[dev]"

# еүҚз«Ҝ
cd ../frontend
npm install
```

### й…ҚзҪ®

```bash
cp backend/.env.example backend/.env
# зј–иҫ‘ .env вҖ” и®ҫзҪ®дҪ зҡ„ LLM API Key
```

### еҗҜеҠЁ

```bash
# з»Ҳз«Ҝ 1: еҗҺз«Ҝ
cd backend && uvicorn app.main:app --reload --port 8000

# з»Ҳз«Ҝ 2: еүҚз«Ҝ
cd frontend && npm run dev
```

жү“ејҖ **http://localhost:3000** еҚіеҸҜдҪҝз”ЁгҖӮ

### иҝҗиЎҢжөӢиҜ•

```bash
cd backend && python -m pytest tests/ -v
```
